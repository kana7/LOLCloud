Découverez ci-dessous les différentes procédures d'installations du système LOL Cloud sur votre machine qu'elle soit sous Mac OS, Windows, Android ou encore IOS.

Veuillez sélectionner votre platforme :

  1. Allez sur www.internet.lu
  2. À droite, utilisez le login pour accéder à LOL CLOUD :
  1. Téléchargez les pilotes du service LOLCLOUD !
  2. Installer le fichier Téléchargé et confirmez le redémarrage de l'ordinateur :
  1. Après redémarrage, cliquez sur la petite flêche en bas à droite de votre Windows et double-cliquez sur le service "LOL Disk Service", puis dans la nouvelle fenêtre, cliquez sur "Add".
  1. Remplissez les champs de la fenêtre suivante avec les données reçues. (Si vous vous trouvez derrière un serveur Proxy, il faut également remplir la partie correspondante. En cliquant OK, vos donnés sont enregistrées et un message confirme le bon fonctionnement du service.
  1. Vous pouvez maintenant utiliser l'espace en ligne comme un répertoire sur votre ordinateur.
    Si vous voulez ajouter le répertoire à vos favoris, il suffit d'aller dans le répertoire, puis faites clique-droit sur favoris et cliquez sur "Add current location to favorites".
  1. Ouvrez l'outil "Finder", puis choisissez l'option "Se connecter au serveur" dans le menu "Aller" en haut" :
  1. Configurez le serveur :
  1. Vous pouvez maintenant utiliser l'espace en ligne comme un répertoire sur votre ordinateur.

Sous Android, le service LOLCLOUD est disponible via une application officielle.

  1. Autoriser votre système android à installer des applications venant de sources inconnues: paramètres > sécurité > sources inconnues
  2. Télécharger l'application LOLCLOUD sur votre téléphone android: LOLCLOUD android App.
  3. Lancer l'installation de l'application.

Sous iOS (iPhone,iPad,...), le service LOLCLOUD peut être utilisé avec une application "WebDAV". Une variété d'applications "WebDAV" existe dans le "Appstore", le marché des applications Apple. Exemples: AirFile, OverTheAir, WebDAV Nav, ...

Le guide d'utilisation suivant se base sur l'application gratuite ou payante "AirFile" :

  1. Téléchargez et installez l'application dans le marché Apple "Appstore" ! L'application apparaîtra sur votre écran.
  1. Ouvrez l'application et configurez votre compte avec les paramètres fournis :
  1. Utilisez l'application pour télécharger/copier/effacer des fichiers/dossiers sur votre LOLCLOUD.
  1. Utilisez l'application pour faire une sauvegarde des photos.
