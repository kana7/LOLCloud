Découverez ci-dessous les différentes procédures d'installations du système LOLCLOUD sur votre machine qu'elle soit sous Mac OS, Windows, Android ou encore IOS.

Veuillez sélectionner votre plateforme :

  1. Vous pouvez vous connecter à votre compte LOLCloud directement via le lien http://lolcloud.internet.lu/
  2. Depuis le site du LOLCloud, vous pouvez y accéder en cliquant sur "Se connecter" dans la page d'accueil ou en haut à droite, dans la barre de menu du site.
  3. Entrez ensuite l'identifiant ainsi que le mot de passe (qui vous ont été fournis) dans les champs respectifs pour vous connecter.
  1. Téléchargez les pilotes du service LOLCLOUD !
  2. Installer le fichier Téléchargé et confirmez le redémarrage de l'ordinateur :
  1. Après redémarrage, cliquez sur la petite flêche en bas à droite de votre Windows et double-cliquez sur le service "LOL Disk Service", puis dans la nouvelle fenêtre, cliquez sur "Add".
  1. Remplissez les champs de la fenêtre suivante avec les données reçues. (Si vous vous trouvez derrière un serveur Proxy, il faut également remplir la partie correspondante. En cliquant OK, vos donnés sont enregistrées et un message confirme l'ajout de votre compte LOLcloud.
  1. Une fois votre compte enregistré avec succès, sélectionnez celui-ci dans votre liste des espaces LOLCloud.
    Cliquez ensuite sur le bouton "Mount" pour indiquer à Windows d'utiliser ce Cloud comme espace de stockage sur votre ordinateur.
  1. Vous pouvez maintenant utiliser l'espace en ligne comme un répertoire sur votre ordinateur.
    Si vous voulez ajouter le répertoire à vos favoris, il suffit d'aller dans le répertoire, puis faites clique-droit sur favoris et cliquez sur "Add current location to favorites".
  1. Ouvrez l'outil "Finder", puis choisissez l'option "Se connecter au serveur" dans le menu "Aller" en haut" :
  1. Configurez le serveur :
  1. Vous pouvez maintenant utiliser l'espace en ligne comme un répertoire sur votre ordinateur.

Sous Android, le service LOLCLOUD est disponible via une application officielle.

  1. Autorisez votre système android à installer des applications venant de sources inconnues: paramètres > sécurité > sources inconnues
  2. Téléchargez l'application LOLCLOUD sur votre téléphone android: LOLCLOUD android App.
  3. Lancez l'installation de l'application.

Sous iOS (iPhone,iPad,...), le service LOLCLOUD peut être utilisé avec une application "WebDAV". Une variété d'applications "WebDAV" existe dans le "Appstore", le marché des applications Apple. Exemples: AirFile, OverTheAir, WebDAV Nav, ...

Le guide d'utilisation suivant se base sur l'application gratuite ou payante "AirFile" :
