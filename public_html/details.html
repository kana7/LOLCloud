Le service stockage en ligne de Luxembourg Online vous permet de stocker et partager des documents, images, vidéos dans un espace en ligne, accessible de partout

UTILISATION : NAVIGATEUR

Le service est accessible à partir d'un navigateur.

Utilisation dans navigateur : LOLCLOUD via navigateur

UTILISATION : POSTE DE TRAVAIL

En configurant le service LOLCLOUD, vous pouvez utiliser l'espace en ligne comme un répertoire sur votre ordinateur.

Utilisation sous Windows : Installation Windows
Utilisation sous MacOs : Installation MacOS

UTILISATION : TÉLÉPHONE MOBILE

Sous iOS, via une application de gestion de fichiers (client WebDav) , vous pouvez télécharger ou consulter les fichiers sur votre stockage en ligne. Vous avez la possibilité de faire des backups immédiats des photos prises avec votre téléphone.

Utilisation sous iOS : Installation iOS

Sous Android, il vous suffit de télécharger l'application LOLCLOUD officielle via le lien ci-dessous.
Ajouter ensuite votre compte pour accéder automatiquement à vos fichiers.

Utilisation sous Android : Installation Android
